# VendingMachine

A Python vending machine application that simulates a beverage dispensing system.

## Features

- Interactive menu for beverage selection
- Multiple drink options with different prices
- Payment and change calculation
- User-friendly command-line interface

## Installation

Install the project dependencies using UV:

```bash
uv sync
```

## Usage

Run the application:

```bash
uv run main.py
```

Or directly with Python:

```bash
python main.py
```

## Project Structure

```
src/
  bobamachine/
    - machine.py      Main vending machine logic
    - __init__.py
main.py              Application entry point
```

## Requirements

- Python 3.8+
